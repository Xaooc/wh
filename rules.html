Справочник по специальным правилам

Этот раздел содержит описания универсальных и уникальных правил оружия Kill Team. Их можно использовать для всплывающих подсказок на листах оперативников и снаряжения. Основной список находится ниже в разделе «Special Rules» и включает как стандартные правила из основного набора, так и специфичные для Новитий Адепты Сороритас.

Special Rules

Каждое правило содержит список псевдонимов в атрибуте data-trait-aliases. Эти псевдонимы используются скриптом для поиска соответствия в таблицах оружия. Для правил с переменными значениями используются плейсхолдеры вида {{value}}.

Дальность (Range)

Оружие с этим параметром может выбирать цели только в пределах указанной дистанции, измеряемой в дюймах. Значение подставляется в запись «Дальность X"».

Поток (Torrent)

После выбора основной цели вы можете выбрать всех прочих видимых врагов в пределах указанной дистанции от неё. Для каждой цели повторяется та же последовательность бросков атаки.

Перегрев (Hot)

Если при броске кубов атаки выпадает хотя бы одна «1», после завершения последовательности атак эта модель получает 3 смертельных раны.

Смертоносность (Lethal)

Значение в записи «Смертоносность X+» показывает минимальный бросок атаки, который считается критическим.

Пронзающий (Armour Penetration)

Это правило уменьшает количество кубов защиты, доступных цели при броске, на величину X из записи «Пронзающий X».

Пронзающие криты (Piercing Crits)

Когда атакующий сохраняет критический успех, цель должна отменить столько собственных успехов защиты, сколько указано в значении правила.

Опустошающий (Devastating)

При критическом успехе цель получает дополнительный смертельный урон, равный значению правила, до применения обычного урона оружия.

Бесшумный (Silent)

Оружие с этим правилом можно использовать, даже если у оперативника приказ «Скрытность» (Conceal). Оно не раскрывает его позицию при стрельбе.

Жестокий (Brutal)

Вражеский оперативник может блокировать удары из этого оружия только критическими успехами. Обычные успехи защиты игнорируются при парировании.

Неустанность (Ceaseless)

При броске кубов атаки можно перебросить все результаты «1».

Рассечение (Rending)

Когда вы сохраняете критический успех, можете также сохранить один обычный успех как критический.

Шок (Shock)

После того как цель получает урон от критического успеха, уменьшите её характеристику APL на 1 до конца ближайшей активации (минимум до 1).

Оглушение (Stun)

При критическом успехе цель теряет 1 APL (до минимума 1), а если атака была в ближнем бою, она также лишается одного куба защиты при этой последовательности.

Суровый (Severe)

Каждый критический успех наносит цели 1 смертельную рану в дополнение к обычному урону.

Насыщение (Saturate)

Оружие заливает область огнём: после разрешения атаки по основной цели нанесите 1 смертельную рану каждому другому вражескому оперативнику, которого вы выбрали в пределах расстояния правила «Поток».

Рипост (Riposte)

Каждый раз, когда вы парируете критическим успехом в ближнем бою, после отмены куба соперника нанесите ему урон, равный обычному урону оружия.

Противопсихер (Anti-PSYKER)

При атаке по цели с ключевым словом PSYKER увеличьте обе характеристики урона на 1 и применяйте правило «Смертоносность 5+».

Ярость праведника (Zealous Rage)

Пока оперативник дерётся этим оружием, оно получает правило «Неустанность» (Ceaseless).

Справочник правил Kill Team подготовлен для удобной навигации и быстрой проверки эффектов.
